## Getting Started

### Prerequisites

Use the Yarn Package Manager

- Install Yarn

```sh
 npm install -g yarn
```

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/sirajeddineaissa/github-repo-search
   ```
2. Install dependencies
   ```sh
   yarn install
   ```
